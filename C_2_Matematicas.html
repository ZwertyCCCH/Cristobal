📖 CONCEPTO BÁSICO

¿Qué es el álgebra?

Definición

El álgebra es la rama de las matemáticas que generaliza la aritmética usando letras (variables) para representar números.

Lenguaje algebraico

Es la forma de escribir expresiones matemáticas usando números, letras y signos de operaciones.

Ejemplo: En lugar de escribir "un número más 5", escribimos x + 5

📊 TÉRMINOS ALGEBRAICOS

¿Qué es un término algebraico?

Partes de un término

Un término algebraico puede ser un número, una letra o una combinación de ambos mediante multiplicación y/o división.

3x²y = 3 · x² · y
Coeficiente · Factor literal

Grado de un término

El grado es la suma de los exponentes de las variables.

5x² → grado 2
x³y → grado 4 (3+1)
x²y³z → grado 6 (2+3+1)
📊 CLASIFICACIÓN

Tipos de expresiones algebraicas

Monomio

1 término

x, y, xy, 3x, 2x²y

Binomio

2 términos

x + y, x - y, 3x + 5y²

Trinomio

3 términos

2x + 3y - 4z, x² + 3x - 7

Polinomio

2 o más términos

x + 3y, x² - y² + z³ + 1

Nota: Un monomio con coeficiente 0 es el término nulo. No se considera un término.
➕ REDUCCIÓN

Términos semejantes

¿Qué son?
Son términos que tienen igual factor literal (mismas letras con mismos exponentes).

✅ Ejemplos de términos semejantes

3x² y 5x²

-2xy³ y 7xy³

4a²b y -a²b

❌ NO son semejantes

3x² y 3x³ (diferente exponente)

5xy y 5x²y (diferente exponente)

2ab y 2ac (diferente letra)

Reducción: Se suman o restan los coeficientes y se mantiene el factor literal.
3x + 5x = 8x   7y² - 2y² = 5y²
📝 OPERACIONES

Eliminación de paréntesis

Signo + antes del paréntesis

Se elimina el paréntesis sin cambiar los signos.

a + (b + c) = a + b + c
(3x + 2) + (4x - 1) = 3x + 2 + 4x - 1 = 7x + 1

Signo - antes del paréntesis

Se elimina el paréntesis cambiando los signos de los términos dentro.

a - (b + c) = a - b - c
(5x - 3) - (2x + 4) = 5x - 3 - 2x - 4 = 3x - 7
✖️ OPERACIONES

Multiplicación de monomios

Regla: Se multiplican los coeficientes y se multiplican los factores literales (sumando exponentes de igual base).

Ejemplo 1

(3a²b³) · (2a⁵c) = 6a⁷b³c

3 × 2 = 6 (coeficiente)
a² × a⁵ = a⁷
b³ se mantiene
c aparece

Ejemplo 2

( -5x³y) · (2xy²) = -10x⁴y³

-5 × 2 = -10
x³ × x = x⁴
y × y² = y³

Ejemplo 3

(2x²) · (3x³) = 6x⁵

Ejemplo 4

(4ab) · (5a²b³) = 20a³b⁴

Recuerda: x = x¹ (exponente 1 invisible)
📊 EVALUACIÓN

Evaluar una expresión algebraica

¿Qué significa evaluar?
Sustituir las variables por números y calcular el resultado.

Ejemplo 1

Evaluar 3x + 5 para x = 2

3(2) + 5 = 6 + 5 = 11

Ejemplo 2

Evaluar x² - 4x + 3 para x = 1

1 - 4 + 3 = 0

Ejemplo 3

Evaluar 2x + 3y para x = 3, y = 2

2(3) + 3(2) = 6 + 6 = 12

Ejemplo 4

Evaluar a² + b² para a = 4, b = 3

16 + 9 = 25

➕ EJEMPLOS

Más ejemplos de reducción

Ejemplo 1

Reducir: 5x + 2y - 3x + 7y

Paso 1: Agrupar semejantes: (5x - 3x) + (2y + 7y)

Paso 2: Sumar: 2x + 9y

Ejemplo 2

Reducir: 4a² + 3ab - a² + 5ab - 2b²

Agrupar: (4a² - a²) + (3ab + 5ab) - 2b²

Resultado: 3a² + 8ab - 2b²

Ejemplo 3

Reducir: (3x + 4) + (2x - 1) - (x + 5)

Eliminar paréntesis: 3x + 4 + 2x - 1 - x - 5

Agrupar: (3x + 2x - x) + (4 - 1 - 5)

Resultado: 4x - 2
